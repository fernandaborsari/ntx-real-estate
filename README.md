## Projects

### 🏡 [NTX Real Estate Market Analysis](./projects/north_texas_housing/)

**End-to-end analytics platform** for the North Texas housing market.

- Modular Python ETL pipeline ingesting Redfin + Zillow public data
- 18 cities · 169 months of price history · ZIP drilldown · 5 bedroom tiers
- Interactive React dashboard: market pulse, performance ranking, ZIP drilldown, price forecasting
- Statistical projections using CAGR + 80% confidence intervals

`Python` `Pandas` `React` `Recharts` `Tailwind CSS` `ETL` `Time Series`

### 🎓 [Academic Stress — ETL Pipeline](./projects/academic_stress/)

**ETL pipeline** processing a student academic stress dataset from Kaggle.

- Data extraction, cleaning, and standardization using Python and Pandas
- Structured output ready for SQL-based analysis and reporting

`Python` `Pandas` `SQL` `Jupyter Notebook` `Data Cleaning` `ETL`
